# Eaglercraft-server-help
How to create an eaglercraft server on eagler.host
Step 1 - visit Https://eagler.host
Step 2 - earn credits (We recomend the 30 minute playtime for infinite credits)
step 3 - create a server - chose version, plugins, gamemodes, rules, etc
step 4 - go to eaglercraft and chose multiplayer
step 5 - chose add server and type the address and name
